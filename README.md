### `upload-image-app`
This is an AWS Amplify web app using the React framework which takes an input photo and uploads it to the cloud (AWS S3 storage).

A couple of resources were used which were super helpful for getting started with this:
- [Amplify Gen 1 Tutorial by Tiny Techical Tutorials](https://youtu.be/ma1FA2be8Ac?si=lO_EASkhXhf25VF4)
- [Amplify Dev Center Documentation](https://ui.docs.amplify.aws/)
